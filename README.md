## Sobre mim

Olá! Sou **Alexandre**, desenvolvedor **Full-Stack** focado na stack **JavaScript/TypeScript**.  
Gosto de construir APIs robustas, seguras e escaláveis no back-end e interfaces modernas, responsivas e com boa experiência de usuário no front-end.

Atualmente meu foco está em:
- APIs RESTful com **Node.js + Express**
- Autenticação JWT, refresh tokens, validação com **Yup/Zod**
- Bancos relacionais (**PostgreSQL + Sequelize**) e NoSQL (**MongoDB + Mongoose**)
- Front-end com **React**, componentização, estados globais e integração com APIs
- Boas práticas: clean code, arquitetura em camadas, versionamento Git profissional, testes
